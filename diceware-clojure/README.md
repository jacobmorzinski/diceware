# diceware

An implementation in Clojure.

## Usage

    $ lein run

or

    $ lein uberjar
    $ java -jar ./target/uberjar/diceware-0.2.0-standalone.jar

or using deps.edn

    $ clj -A run

The deps.edn uberjar is not yet standalone; todo: review https://github.com/tonsky/uberdeps
